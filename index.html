Política de Privacidade – Audit+

Última atualização: 27 de julho de 2025

1. Coleta de Dados

O aplicativo coleta dados para fins estatísticos e de análise relacionados a infecções hospitalares. Todos os dados são coletados de forma agregada e não contêm informações pessoais ou sensíveis que possam identificar pacientes individualmente.

2. Uso dos Dados

Os dados são utilizados exclusivamente para:

  • Geração de relatórios estatísticos para unidades hospitalares
  • Apoio à decisão clínica em casos de infecções
  • Visualização de indicadores por setor e unidade funcional

3. Dados dos Usuários

Coletamos apenas as informações necessárias para identificar o perfil e função profissional do usuário (como tipo de cargo ou setor). Esses dados são utilizados para garantir os níveis corretos de acesso ao sistema.

4. Permissões do Dispositivo

O aplicativo pode solicitar permissões para:

  • Câmera: para capturar evidências visuais de casos clínicos
  • Arquivos: para anexar documentos ou imagens relacionadas aos processos
  • Localização: para contextualizar registros por setor físico da unidade hospitalar

Essas permissões são utilizadas exclusivamente para melhorar a integridade e precisão das informações, e nunca para rastrear ou identificar usuários.

5. Compartilhamento de Dados

Os dados não são compartilhados com terceiros. Todos os registros permanecem armazenados de forma segura nos servidores vinculados à unidade hospitalar responsável pelo uso do aplicativo.

6. Segurança

Empregamos práticas técnicas e organizacionais adequadas para proteger os dados contra acesso não autorizado, uso indevido ou divulgação.

7. Contato

Em caso de dúvidas, entre em contato com o responsável pelo app ou pela unidade hospitalar onde o sistema é utilizado.
